# 💰 Expense Management System

A simple and efficient **Command-Line Expense Management System** built using Python.  
This project helps users **track, manage, and update daily expenses** using file handling with CSV storage.

---

## 🚀 About the Project

Managing daily expenses is an essential part of financial discipline. This project provides a **lightweight and beginner-friendly solution** to record and manage expenses directly from the terminal.

It demonstrates core programming concepts like:
- Object-Oriented Programming (OOP)
- File Handling
- Data manipulation using CSV

---

## ✨ Features

- ➕ Add new expenses  
- 📋 View all recorded expenses  
- 🔄 Update existing expenses  
- ❌ Reset all expenses  
- 💾 Persistent data storage using CSV file  

---

## 🛠️ Tech Stack

- **Language:** Python  
- **Concepts Used:** OOP, File Handling  
- **Storage:** CSV (expenses.csv)  

---

## 📂 Project Structure

- Expense → Class representing an expense entry
- main.py → Contains the main program logic and menu system
- expenses.csv → Stores all expense records
- Core functionalities:
  - Add expense
  - View expenses
  - Update expense
  - Reset all data
  - File handling using CSV

---

## ⚙️ How to Run

1. Clone the repository using:  
`git clone https://github.com/ashutoshkashyap04/Expense_Management_System`

2. Go to the project folder:  
`cd Expense_Management_System`

3. Run the program:  
`python main.py`
